### 安裝相關套件

`npm install`  
`git clone -b master https://github.com/jerryc127/hexo-theme-butterfly.git themes/butterfly`

### 自動部屬

`hexo generate`  
`hexo deploy`

### 新增 hexo 專案

`hexo init myblog`

### 開啟伺服器

`hexo server`

### 更換主題

[butterfly](https://butterfly.js.org/posts/21cfbf15/#%E5%AE%89%E8%A3%9D)

### 自動部屬工具

` npm install --save hexo-deployer-git`

### 新增頁面

`hexo new title`
