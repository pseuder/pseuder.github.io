### 安裝 hexo

`npm install -g hexo-cli`

### 新增 hexo 專案

`hexo init myblog`

### 開啟伺服器

`hexo server`

### 更換主題

[butterfly](https://butterfly.js.org/posts/21cfbf15/#%E5%AE%89%E8%A3%9D)

### 自動部屬工具

` npm install --save hexo-deployer-git`

### 自動部屬

`hexo generate`  
`hexo deploy`

### 新增頁面

`hexo new title`
