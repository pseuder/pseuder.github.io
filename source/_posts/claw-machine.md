# 專案描述

軟體工程期末專案, 用 three.js & cannon.js 打造線上夾娃娃機平台

# 使用技術

- 前端框架：vue2
- 後端: python
- 應用框架:Django
- 3D 模型：three.js
- 物理引擎：cannon.js
- 資料庫：db.sqlite3
- 託管平台：pythonanywhere
