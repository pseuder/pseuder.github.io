# 專案描述

能快速產生並方便使用, 使用直覺的報價單

# 使用技術

- 前端: vue2
- UI 組件: Element UI
- Router: vue-router
- HTML 轉換 PDF: vue-html2pdf
- 手繪版簽名: vue-esign
- 後端: Django, Python
- 資料庫: sqlite
