專案描述

此網站為面向公司內部人員的活動報名系統，公司內部人員可以建立與參加自訂活動

使用技術

  • UI 規劃: Figma

  • 前端: vue3, vuex(全域管理), vite(快速建構)

  • 後端: express

  • 資料庫: mongoose

  • Router: vue-router

  • UI 組件: Ant Design

  • CSS 框架: Tailwindcss

  • 多國語系: vue-i18n

  • 身份驗證: jwt, reCAPTCHA

  • 資料驗證: vee-validate, joi

  • 第三方登入: vue3-google-login

  • 開發環境: eslint, prettier

  • 部署平台: render

  • RWD

功能介紹

  • 登入畫面
  • 註冊畫面
  • 瀏覽活動, 標籤過濾, 條件排列, 活動狀態, 加入最愛
  • 變更排列模式
  • 活動詳細資訊, 圖片輪播
  • 建立活動
  • 建立活動錯誤
  • 查看跟自己有關的活動(已報名, 已收藏, 已建立)
